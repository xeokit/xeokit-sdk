LASLoaderPlugin & TreeViewPlugin

Making a LiDAR scan appear in the TreeViewPlugin

The LASLoaderPlugin's elementId parameter causes the entire model to be loaded into a single Entity that gets this ID.

In conjunction with that parameter, we can also use the LASLoaderPlugin's metaModelJSON parameter to create a MetaModel with a MetaObject that corresponds to that Entity.

The TreeViewPlugin is then able to have a node that represents the LiDAR model and controls the visibility of that Entity.
