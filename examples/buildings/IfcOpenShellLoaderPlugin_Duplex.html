IFCOpenShellLoaderPlugin

Loading an IFC File

IFCOpenShellLoaderPlugin is the easiest way to load IFC models into a xeokit Viewer.

IFCOpenShellLoaderPlugin loads IFC STEP files and parses them within the browser using ifcopenshell.api, to create 3D objects within the Viewer.

Limitations

Loading and parsing huge IFC STEP files can be slow, and can overwhelm the browser, however. To view your largest IFC models, we recommend instead pre-converting those to xeokit's compressed native .XKT format, then loading them with XKTLoaderPlugin.
