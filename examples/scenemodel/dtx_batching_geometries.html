SceneModel

SceneModel with various batched geometries

In this example, we're programmatically building a simple model using xeokit's SceneModel component, while demonstrating the various types of geometry primitive that are available, using batched geometry for each object.

Reusing geometry speeds up rendering and reduces the memory footprint.

Components Used

  • Viewer
  • SceneModel
  • buildBoxGeometry
  • buildBoxLinesGeometry
  • buildCylinderGeometry
  • buildGridGeometry
  • buildSphereGeometry
  • buildVectorTextGeometry
  • buildTorusGeometry
