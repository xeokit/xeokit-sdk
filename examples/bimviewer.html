xeokit / BIMViewer Examples



BIMViewer is an open source 2D/3D BIM viewer that runs in the browser and loads models from your file system. The viewer is built on xeokit, and is bundled as an example application within the xeokit SDK. Click on one of the links below to try it out, or try some of our SDK programming examples to learn about how you can build your own custom applications.
