First-person mode with pointer lock

In this example, we're loading a double-pecision BIM model from the file system, then navigating it using first-person mode. We'r also enabling pointer locking, which updates the camera without moving the pointer and risking losing it in a corner of the screen.

This model is centered at [1842022, 10, -5173301].
