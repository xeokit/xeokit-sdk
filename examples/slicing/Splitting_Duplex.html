Model Splitting

Demonstrates how model entities can be split into new entities

This example demonstrates how the math.makeSectionPlaneSlicer function is used to generate a split of an existing mesh.
The result submeshes are added to the scene, and the original mesh - with no modification - is being hidden.

Example workflow

  1. Select an entity (e.g. the green roof) by left-clicking on it
  2. Press the [split 1] button
  3. Temporarily hide/disable the SectionPlane with the [Sect. plane off] bytton
  4. Note two new split subentities under the "Basic Roof:Live Roof over Wood Joist Flat Roof:184483" node in the tree view.
  5. Right click the "Basic Roof:Live Roof over Wood Joist Flat Roof:184483" tree view node, and click the [Unsplit] menu item to revert the split.
