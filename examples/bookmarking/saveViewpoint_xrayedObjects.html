Saving a BCF Viewpoint Containing X-Rayed Objects

The BIM Collaboration Format (BCF) is a structured file format used for issue tracking with a building information model. BCF defines views of a building model and associated information on collisions and errors connected with specific objects in the view.

In this example, to simulate logging a model issue via BCF, we're loading a BIM model from the file system, slicing it with a section plane, creating a grid groundplane and a bitmap plan view, selecting some objects, X-raying some objects, then saving the viewer state as a JSON-encoded BCF viewpoint using the BCFViewpointsPlugin xrayAsZeroAlpha option.
