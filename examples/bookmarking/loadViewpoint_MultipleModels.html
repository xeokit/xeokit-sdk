BCFViewpointsPlugin

Loading a BCF viewpoint for multiple copies of a model

The BIM Collaboration Format (BCF) is a structured file format used for issue tracking with a building information model. BCF defines views of a building model and associated information on collisions and errors connected with specific objects in the view.

In this example, we're simulating an unusual use case: loading two copies of the same BIM model from the file system, then loading a JSON-encoded BCF viewpoint to highlight a simulated issue with those two model copies.
