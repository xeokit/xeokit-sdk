Scene Graph

Scene graph with GPU-resident geometry

In this example, we're programmatically building a simple model from xeokit's scene graph components, with geometry stored solely in GPU memory.

We have the option to not retain geometry in browser memory, in case we need a lower footprint.

Components Used

  • Viewer
  • Node
  • Mesh
  • PhongMaterial
  • VBOGeometry
  • buildBoxGeometry
