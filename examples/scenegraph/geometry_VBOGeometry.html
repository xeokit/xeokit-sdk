VBOGeometry

GPU-resident geometry for smaller browser memory footprint

In this example, we're creating a simple 3D scene that contains geometry that is stored on the GPU, but not in browser memory.

Storing the geometry data only on the GPU reduces the browser memory footprint, but means that we cannot access the data in JavaScript.
