    An abstract 3D scene element that can be individually shown, hidden, selected, highlighted, xrayed, culled, picked, clipped and bounded.

    Entity provides an abstract interface through which different concrete types of scene element can be accessed and manipulated uniformly.

    Entities Representing Models

    • An Entity represents a model when Entity#isModel is true.
    • Each model-Entity is registered by Entity#id in Scene#models.
    • Each model-Entity can also have a MetaModel with a matching MetaModel#id, by which it is registered in MetaScene#metaModels.

    Entities Representing Objects

    • An Entity represents an object when Entity#isObject is true.
    • Each object-Entity is registered by Entity#id in Scene#objects.
    • Each object-Entity can also have a MetaObject with a matching MetaObject#id, by which it is registered MetaScene#metaObjects.

    Updating Batches of Objects

    Scene provides the following methods for updating batches of object-Entities using their Entity#ids:

    • Scene#setObjectsVisible
    • Scene#setObjectsCulled
    • Scene#setObjectsSelected
    • Scene#setObjectsHighlighted
    • Scene#setObjectsXRayed
    • Scene#setObjectsEdges
    • Scene#setObjectsColorized
    • Scene#setObjectsOpacity
